ScreenSense: Kids' Screentime Visualization & Analytics 📊📱

ScreenSense is a data analytics pipeline designed to uncover children's digital consumption patterns across demographics, device choices, and health outcomes using the Indian Kids Screentime 2025 dataset.

🚀 Tech Stack
* Data Engineering & ML:** Python (Pandas, NumPy, Scikit-Learn)
* Exploratory Data Analysis:** Matplotlib, Seaborn, Plotly Express
* Business Intelligence:** Tableau Desktop
* Environment:** Jupyter Notebook

📂 Project Modules Implemented
1. Data Preprocessing & Feature Engineering: Cleaned missing values, handled categorical data, and generated age bands.
2. Exploratory Data Analysis (EDA): Evaluated screentime variations across urban/rural divides, gender, and age segments.
3. Predictive Modeling (ML): Trained a Random Forest Classifier to catch children at risk of exceeding safe screen boundaries.
4. BI Dashboarding: Designed a 7-sheet interactive dashboard in Tableau tracking health risk matrices and learning balances.

📊 Core Insights Uncovered
* A major portion of children consistently exceed healthy baseline daily usage limits.
* Handheld devices are the main hardware choices driving high exposure metrics.
* Elevated screen hours are heavily coupled with adverse health conditions like sleep disruptions and high anxiety.
